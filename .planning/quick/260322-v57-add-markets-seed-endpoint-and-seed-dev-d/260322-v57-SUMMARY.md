## What Was Built

**Seed endpoint (Task 1):**
- Added `SEED_MARKETS` const array with 15 market definitions (10 polymarket, 5 kalshi)
- Route `app.post("/seed", ...)` placed before `/:id` route to prevent param collision
- Queries existing `platformId` values, filters to only new markets, inserts them
- After insert, queries back by `platformId` using `inArray` to get auto-increment IDs
- Inserts one price row per market with realistic `yesPrice` (0.12–0.72), computed `noPrice = 1 - yesPrice`, `yesBid/yesAsk = yesPrice ± 0.01`, and volume (48k–450k)
- Returns `{ seeded: N }` where N = 0 on repeat calls (idempotent)

**API and UI (Task 2):**
- `api.seedMarkets()` added to `src/ui/lib/api.ts`
- `api.syncMarkets()` return type updated to include optional `error?: string`
- Markets.tsx gains a `seed` mutation (useMutation wrapping `api.seedMarkets()`)
- Empty state splits on `search`: when search active shows text; when empty shows "No markets synced yet." + "Seed Dev Data" button
- After successful seed shows "Seeded N markets" count
- Sync error string displayed inline after the Sync Markets button
