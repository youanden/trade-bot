  <action>
    Create `src/worker/core/simulation/engine.ts` with a clean dependency injection approach:

    **BacktestClock class:**
    ```typescript
    export class BacktestClock {
      private currentMs: number;
      private readonly intervalMs: number;
      constructor(startTime: string, tickIntervalMs: number);
      now(): string;    // returns ISO-8601
      advance(): void;  // currentMs += intervalMs
      isAfter(isoTimestamp: string): boolean;
    }
    ```

    **BacktestDeps interface (dependency injection):**
    ```typescript
    export interface BacktestDeps {
      createDb: (d1: unknown) => any;
      createExchangeClient: (env: any, platform: string, simFeed?: any) => ExchangeClient;
    }
    ```
    This is the SINGLE DI mechanism. No module-level mutable state, no `_activeDb`/`_activeSimClient` globals. The `runBacktest` function accepts an optional `deps` parameter. In tests, callers pass deps directly. In production use, defaults import from the real modules.

    **Types:**
    ```typescript
    export interface BacktestConfig {
      botType: string;
      botConfig: BotConfig;
      scenario: GeneratedScenario;
      tickIntervalMs?: number;
      platform: "polymarket" | "kalshi";
      virtualBalance?: number;
      mockAI?: { run: (model: string, inputs: unknown) => Promise<unknown> };
    }
    export interface EquitySnapshot { timestamp: string; balance: number; tickIndex: number; }
    export interface BacktestResult { equityCurve: EquitySnapshot[]; tradeCount: number; finalBalance: number; runId: string; }
    ```

    **runBacktest function:**
    ```typescript
    export async function runBacktest(config: BacktestConfig, deps?: BacktestDeps): Promise<BacktestResult>
    ```

    Implementation steps:
    1. Resolve deps: `const resolvedDeps = deps ?? { createDb: (await import("../db/client")).createDb, createExchangeClient: (await import("../exchanges/factory")).createExchangeClient }` — lazy import defaults so tests can pass overrides without mock.module.
    2. Create isolated DB: `const db = resolvedDeps.createDb(null)` — each call gets fresh :memory: SQLite.
    3. Create BacktestClock from `scenario.prices[0].timestamp` with `tickIntervalMs ?? 60_000`.
    4. Create PriceFeed: `new PriceFeed(scenario)`.
    5. Create SimulatedBot: `new SimulatedBot(config.botConfig, db)`.
    6. Seed scenario data: insert `scenario.market` into `markets` table, insert all `scenario.prices` into `prices` table. Insert a `botInstances` row. Set `bot.config.dbBotId` to the returned id.
    7. Create SimExchangeClient via `resolvedDeps.createExchangeClient({} as Env, config.platform, { feed, config: { simulatedNow: () => clock.now(), virtualBalance: config.virtualBalance ?? 1000 } })`. Store as `simClient`.
    8. Build env stub with `DB: db`, `ENVIRONMENT: "backtest"`, `AI: config.mockAI`. The `resolvedDeps.createDb` mock returns `db`, and `resolvedDeps.createExchangeClient` mock returns `simClient` — so when strategies internally call these functions, they get the engine's instances.
    9. Override globalThis.Date for the tick loop so `new Date()` calls inside strategies/PortfolioRisk return simulated time. Restore in `finally` block.
    10. Get strategy: `const strategy = getStrategy(config.botType)`. Throw if undefined.
    11. Tick loop: for each price in scenario, call `strategy(bot as any, env as any)`, snapshot equity via `simClient.getBalance()`, then `clock.advance()`.
    12. Return `{ equityCurve, tradeCount: bot._tradeCount, finalBalance, runId: crypto.randomUUID() }`.

    **Test file `test/core/engine.test.ts`:**
    Tests pass deps directly — no mock.module needed for engine tests because `runBacktest` accepts `BacktestDeps`:

    ```

      createExchangeClient: (_env: any) => _env._simClient,
    }));
    ```
    The engine stores `simClient` on `env._simClient` and passes `db` as `env.DB` so the mocks return them.
