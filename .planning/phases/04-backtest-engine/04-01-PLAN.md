<objective>
Create the SimulatedBot class that duck-types the BaseBotDO interface so strategies can run in backtest mode, and add injectable clock support to PortfolioRisk so multi-day backtests correctly reset the daily loss circuit breaker.

Purpose: Strategies access bot via `(bot as any).config` and `(bot as any).recordTrade(...)`. SimulatedBot satisfies both without importing `cloudflare:workers`. PortfolioRisk currently hardcodes `new Date()` in `isDailyLossBreached()`, which would break multi-day backtests.

Output: SimulatedBot class, updated PortfolioRisk, unit tests for both.
</objective>
