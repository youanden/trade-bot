         g. Compute cost: `fillSize * filledPrice + fee`
         h. If `cost > this.balance`, return {orderId, status: "failed"}
         i. Deduct: `this.balance -= cost`
         j. Upsert position in `this.positions` Map (key: `${marketId}-${outcome}`)
         k. Store and return OrderResult

       - `getOpenOrders(marketId?)`: filter orders map for status==="open"
       - `getPositions()`: return Array.from(this.positions.values())
       - `getBalance()`: return this.balance
